### Automic Car
라즈베리 파이와 안드로이드 앱을 이용하여 자동차 모형을 움직이는 프로젝트 입니다. 
- 언어 및 작업툴: Python, Vscode, Android Studio, Raspberry Pi
- 인원: 5명
- 기간: 2020.11.02 ~ 2020.11.26
- 내용
    - Line Tracking
        앱에서 버튼을 누른 곳으로 이동 (지정된 코스만 가능)
    - Joystick Controller
        조이스틱으로 상시 운행 가능
- 기여: 안드로이드 앱 제작, UDP통신 관련 코드 작성
